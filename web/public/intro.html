Tutti

Test Javascript on different browsers simultaneously

Getting Started

To get started, you first need to create a room and connect your browser to it. Once a room is created you can connect other browsers to it as well.

Then, to go further, follow the instructions for using the driver.

How it Works

When you enter a Javascript command, it is immediately sent to each browser that's connected to the room. Each browser then eval() the Javascript within an iframe sandbox and then send the result back to be displayed.

Communication happens in real-time thanks to the brilliant Socket.IO which puts a nice abstraction around WebSocket and Comet techniques.

Browser Support

  • Firefox 2, 3.6 and 4
  • Chrome 8 - 11
  • Safari 5
  • Safari 4 Mobile for iPad and iPhone
  • Opera 10, 11
  • Internet Explorer 6, 7, 8 and 9
